# AI Sentiment Studio

A full-stack AI-based Sentiment Analysis Web Application built using Python & Flask.

## Features
- User Authentication (Login / Register)
- Beautiful Aesthetic UI
- AI-based Sentiment Analysis
- Personal Dashboard
- Responsive Design (Mobile + Desktop)

## Tech Stack
- Python
- Flask
- TextBlob (NLP)
- SQLite
- HTML/CSS
